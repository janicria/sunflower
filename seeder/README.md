## Sunflower ~ seeder/

This directory contains `seeder`, sunflower's very own build tool! It's structure is as follows

```
sunflower/seeder/ 
   src/main.rs        # Binary root file
   src/cmd.rs         # Used to run commands
   Cargo.toml         # Config file used by cargo   
   README.md          # The file you're reading!
```
