## Sunflower ~ libutil/

This directory contains the `libutil` utilities library used by most of sunflower's components. It's structure is as follows

```
sunflower/libutil/ 
   src/lib.rs         # Library root file
   src/send.rs        # Useful synchronization types
   src/sync.rs        # Types formatting data in weird ways
   Cargo.toml         # Config file used by cargo   
   README.md          # The file you're reading!
```
